## 0. 升级背景 (OLS 代理 → 精确反事实)

首轮用 OLS 回归 D_clean = D - α·r_p (α 估计), 信号 ρ(D,D_clean)=0.7674, exp15 (r_p max) 降 39 位。
但 OLS 有两个缺陷 (用户预警, 本轮验证):
1. **共线性过扣**: D = C(p) + A(p), C(p) 也可能与 r_p 相关 (高频专家贡献大), 回归 α·r_p 混了 C(p)
   的 r_p 相关部分 → 过扣真贡献。A_ols med=0.00398 vs A_causal med=0.00025 (**16× 高估**),
   corr(A_causal, A_ols)=0.3465 (弱) — 定量验证过扣。
2. **循环嫌疑**: 减 α·r_p 后高 r_p 者排名必然下移, exp15 42→3 是循环伪影。

精确反事实: Type A 删 p 闭式塌缩 — 幸存权重膨胀 1/(1-σ_p), σ_p = softmax(logits)[p]。
**质量保持删** = 删 p 召回第 9 + 幸存权重 = baseline softmax 值 (乘回 1-σ_p, 未膨胀)。
两臂: D(p) 裸删 (含膨胀) vs D_restored(p) 质量保持删 (未膨胀) = C(p)。
A(p) = D - D_restored = 逐专家因果塌缩, 零回归零共线性。废 Type B 脏臂 (质量保持删 = 同模型
Type A 分母不含 p 实现, 无需强加 renorm 把 OLMoE 弄成 nll 6.2 残废)。

### 2.2 关键反数据修补: exp15 (OLS 明星案例) 精确版不复现

exp15 r_p=0.0323 (全局最大), OLS 版 rankD=42→rankD_clean=3 (降 39 位, 明星案例)。
**精确版**: D=+0.00155, D_restored=+0.00153, A_causal=+0.00002 (几乎零!), rankD=42→rankDr=44
(**不降反升 2 位**)。

机制 (I): A(p) = D - D_restored = 膨胀效应对 loss 的影响 ∝ σ_p/(1-σ_p) · ⟨∂L/∂y, y⟩。
exp15 σ_p 大 (0.0323) 但 ⟨∂L/∂y, y⟩ (专家组合输出与 loss 梯度对齐) 小 → A_causal≈0。
**证伪"低分高频专家 (高 r_p) 最受污染"的简单主张** — A(p) 不由 r_p 单独决定, ρ(A_causal,r_p)=0.40 印证。
OLS 版 exp15 的 42→3 完全是循环伪影 (减 α·r_p 让高 r_p 必然下移, 但真实 A_causal(exp15)≈0 不应下移)。

### 2.4 噪声威胁补救: 配对 ρ 不构成排序变化证据 (推理修正)

**动机**: ρ(D,D_restored)=0.9322 < 0.95 不能直接当"排序因果变化"证据 — D 与 D_restored 均为 40 窗噪声
估计, 0.95 阈值系拍定无依据。本轮追加 split-half 信度零线 + A(p) 配对 bootstrap CI。

**方法 (纯分析, 复用 per-window 数据)**:
- split-half ρ(D): 40 窗 200 次 random split 分两半 (各 20 窗, 独立), 各半算 per-expert D(p)=mean(drop_p[half]−base[half]), ρ(D_h1,D_h2) 平均 = D 自身信度
- A(p) 配对 bootstrap CI: A_win(p)=drop_win(p)−rest_win(p) (两臂同 token 同窗, base 完全消掉), per-window block bootstrap 1000x, 配对差分消窗口级共同噪声 → CI 应远窄于 D_restored

**结果**:
- split-half ρ(D) = **0.4201 ± 0.0851** (20 窗独立两半信度)
- A 配对 CI: 仅 3/64 专家显著正 (§2.5), exp18 不跨 0 铁证
- A 自身 split-half ρ(A)=0.0601 (近零, 因 61/64 专家 A≈0 rank 噪声大), 但配对 CI 仍有 3 专家显著 — 配对差分消共同噪声后稀疏信号稳

**推理修正 (关键)**: 配对 ρ(D,D_restored) 测的是**共享同一批 40 窗上两臂的一致性** (两臂同窗 → 窗口级噪声共享), 不是"总体专家排序是否改变"; split-half ρ(D) 测的是 20 窗独立两半的信度 (噪声独立)。**两量尺度不同 (一个噪声共享一个噪声独立), 0.93 与 0.42 之间任何方向比较都不可判读**。
- 正确推理: 配对 ρ 本身不构成排序变化的证据 (它不是总体排序变化的检验量) → 0.95 阈值废弃, 全局重排无证据
- **错误推理 (本轮原写, 已撤回)**: "ρ 远高于天花板 → 证伪"。这犯了与被证伪原主张同类的尺度错误 — 把两个不同尺度的 ρ 直接比方向。审稿人会指出此句与原 ρ<0.95 主张犯同类错误
- 结论不变: 0.95 阈值废弃, 全局重排无证据; 但理由从"高于天花板"改为"配对 ρ 非排序变化检验量"

**判读**: 上一轮"ρ=0.93<0.95 因果排序变化"headline 废弃 (配对 ρ 非排序变化检验量)。headline 换锚为 A(p) 配对 CI 稀疏显著。

### 2.6 副结果: 剪枝判据信度诊断 (split-half 应为必报诊断, tier E)

**无一报告判据信度**。**split-half 信度应作为剪枝论文必报诊断** — 此主张打的是 7.1 瞄准的同一批文献,

- 证据等级 E (相关性观察: 校准集规模下判据信度低), 独立于主结果 (A(p) 稀疏因果, C), 可单独成立
- 扩窗目标精确化: 信度 0.9 需 ≥250 窗 (非 1000, 1000 浪费), 指导后续扩窗实验取 ≥250

## 3. 机制结论

1. **因果塌缩分离成功 (C)**: 两臂唯一差异 = 膨胀因子 1/(1-σ_p), A(p)=D-D_restored 逐专家因果精确,
   零回归零共线性。A(p) 因果识别成立 (反事实设计), 但稀疏: 仅 3/64 专家 A 配对 CI 显著正 (§2.5)。
   **配对 ρ(D,D_rest)=0.93 非排序变化检验量** (测共享窗口两臂一致性, 与 split-half ρ(D)=0.42 尺度不同不可比), 0.95 阈值废弃 (§2.4)。
6. **剪枝判据信度诊断 (副结果, E)**: 40 窗 Spearman-Brown 信度 ~0.59, r₁≈0.035/窗, 信度 0.9 需 ~250 窗。
   REAP/统一打分框架/EAC-MoE 未报判据信度 → split-half 信度应作为剪枝论文必报诊断, 解释 54/64 CI 跨 0 (§2.6)。
2. **OLS 代理系统性高估 (E)**: A_ols med=0.00398 vs A_causal med=0.00025 (16×), corr=0.35 —
   OLS 共线性过扣真贡献定量验证。OLS 版 ρ=0.77, rank_shift 39, exp15 42→3 均含循环水分。
3. **r_p 非塌缩唯一决定 (I)**: ρ(A_causal,r_p)=0.40 中等。exp15 r_p max 但 A≈0。A(p) ∝ σ_p/(1-σ_p)·⟨∂L/∂y,y⟩,
   r_p 是其一因子, 绝对形式需含梯度对齐项 (future work ③ 印证)。
4. **多数专家真实贡献与零不可分 (E)**: D_restored 54/64 CI 跨 0, 24/64 为负。中位 +0.00027 正但 CI 跨 0。
   撤回首轮"很多重要专家其实有害"表述 (caveat①) — 只能说"多数专家真实贡献与零不可分, 塌缩伪影
   使其 D 显著正"。剪枝判据 D 高估的是"显著正"假象, 真实 C(p) 多数与零不可分。
5. **Type B 脏臂正确废弃 (E+I)**: 质量保持删 = 同模型 Type A 分母不含 p 实现, 无需强加 renorm 失配。
   原 Type B 零结果未守 (ρ_B=0.86) 源于 renorm 重组效应 (非分母塌缩), law A(p)=0 机制未被证伪。

  A(p) = D - D_restored = 逐专家因果塌缩 (闭式, 零回归)。
- 证据 **C** (因果反事实: 两臂唯一差异 = 膨胀因子, do(膨胀) vs do(不膨胀), 非统计分离) + I (law 闭式推导)。
- bootstrap: per-window block 1000x, D_restored_ci + A_ci (配对, 两臂同 token 同窗差分消窗口级共同噪声)。
- **主张 (换锚)**: 剪枝判据污染因果实在但稀疏 (3/64 专家 A 配对 CI 显著正, exp18 不跨 0 铁证, 单专家可达
  实测损伤 84%), 修正闭式免费 (C)。全局 ρ(D,D_restored)=0.93 **远高于** split-half 信度天花板 0.42 → ρ 排序变化
  主张被噪声零线证伪, 降为辅助统计; 0.95 阈值弃用。弱化: A/D 中位 17% (非 84%, 84% 为 exp18 单例)。

## 5. 反数据修补

三轮迭代, 每轮反杀上一轮 (写入论文附录作"去污染判据为何用闭式反事实而非回归"的最有力论证):
1. **OLS 代理版**: ρ(D,D_clean)=0.7674, exp15 42→3 明星案例 — 含 OLS 共线性循环伪影 (减 α·r_p 让高 r_p 必然下移)
2. **精确反事实版**: ρ(D,D_restored)=0.9322, exp15 不复现 (A_causal≈2e-5), OLS 16× 高估定量, exp15 撤回 —
   但用 ρ<0.95 宣称"排序因果变化", 0.95 阈值拍定无依据
3. **噪声补救轮 (本轮)**: split-half ρ(D)=0.4201 (20 窗独立两半信度) + 配对 ρ(D,D_rest)=0.93 (共享窗口两臂一致性, 尺度不同不可比方向) → **配对 ρ 非排序变化检验量, 0.95 阈值废弃, 全局重排无证据** (推理修正: 非"ρ 高于天花板故证伪", 那是同类尺度错误);
   A(p) 配对 CI 救出稀疏因果主张 (3/64 显著, exp18 铁证); 白捡副结果 split-half 信度诊断 (40 窗 SB~0.59, 应为剪枝论文必报诊断)。诚实两侧: ρ headline 废弃不掩盖; A(p) 因果识别仍 C 级成立但稀疏不夸大。
